# AdminLTE RTL AND LTR

#Usage
Install node_modules:
```bash
$ npm install 
```
Install Grunt:
```bash
$ npm install -g grunt-cli
```
Compile files:
```bash
$ grunt dist
```

####AdminLTE RTL(build/less/variable.less):
---
```html
@tag-align: right;
@left:      right;
@right:     left;
```
####AdminLTE LTR(build/less/variable.less):
---
```html
@tag-align: left;
@left:      left;
@right:     right;
```
